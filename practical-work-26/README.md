# Практическая работа №26: Анализ PHP и знакомство с Laravel

## Описание проекта
Этот проект демонстрирует переход от legacy PHP к современному фреймворку Laravel.

## Выявленные проблемы в legacy-коде

### 1. Проблемы безопасности
- Уязвимость к SQL-инъекциям через прямые $_GET запросы
- Отсутствие валидации и санитизации данных
- Прямая работа с суперглобальными массивами

### 2. Архитектурные проблемы
- Смешение PHP логики и HTML представления
- Отсутствие разделения ответственности
- Процедурный подход вместо объектно-ориентированного

### 3. Проблемы сопровождения
- Отсутствие обработки ошибок
- Жестко закодированные настройки
- Сложность тестирования и расширения

## Установка и запуск проекта

### Требования
- PHP 8.0+
- Composer
- Расширения PHP: mbstring, xml, ctype, json

### Установка
```bash
composer install
cp .env.example .env
